# Quiz Game Application

##  Task 03 (Alternative)

An interactive multiple-choice quiz game that displays questions, collects user answers, and shows the final score at the end. The application is built using basic web technologies and provides a simple, user-friendly interface.

---

##  Features
- Interactive multiple-choice questions
- Real-time answer selection
- Automatic score calculation
- Final score display
- Restart quiz functionality
- Clean and responsive user interface

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

##  Project Structure
quiz-game/
│
├── index.html

---

##  How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. The quiz will open in your default web browser

---

##  How It Works
- The application displays one question at a time
- Users select an answer from the given options
- Each correct answer increases the score
- At the end of the quiz, the final score is displayed
- Users can restart the quiz if they want to try again
